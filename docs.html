Bot Setup:

1. প্রথমে তোমার GitHub অ্যাকাউন্ট দিয়ে Section ID জেনারেট করো।

2. সেই ID দিয়ে বট কানেক্ট করো Panel থেকে।

3. .env ফাইল কনফিগার করে নাও। (Bot name, Owner number, Session etc.)
