Lovlyboy-x1 Deployment & Setup Guide

1. Fork the Repository

Go to Lovlyboy-x1 GitHub and click on Fork to make a copy under your GitHub account.

2. Edit Configuration

In your forked repository, go to config.js and .env files and replace with your own:

  • OWNER_NUMBER: Your WhatsApp number with country code
  • SESSION_ID: Your session string
  • GIT_SECTION_ID: A unique name (ex: samin-001)

3. Deploy Bot

You can deploy your bot using the following platforms:

  • Heroku (Recommended)
  • Replit

4. Connect Device (Section ID)

Go back to the Panel and enter your Section ID to connect your device.

5. Features & Plugins

Once connected, explore the bot features including:

  • Auto-react
  • Sticker generation
  • Video mention
  • Music player
  • Group & Admin tools

Advanced plugin system will be available soon!
