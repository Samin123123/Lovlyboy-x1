LovlyBoy X1 Deployment Guide

Follow these steps to set up your bot:

  1. Fork the GitHub repo: Lovlyboy-x1
  2. Go to your fork → Click "Settings" → "Secrets and variables" → "Actions"
  3. Add the following secrets:
    • OWNER_NUMBER
    • SESSION_ID
    • BOT_NAME
  4. Click "Actions" tab → Select "Deploy Bot" → Run workflow manually
