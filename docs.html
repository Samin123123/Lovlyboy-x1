Lovlyboy-x1 Bot Documentation

Setup Instructions

  1. Clone the repository or download the ZIP file.
  2. Install all dependencies: npm install
  3. Start the bot: npm start

Session Creation

Open session.html from the homepage and scan QR code to generate session.

Or, use WhatsApp Number Login to link easily.

Deployment Options

  • Deploy on Railway
  • Deploy on Heroku
  • Deploy on Replit
  • Deploy on Render

Each platform has simple one-click deploy buttons available.
